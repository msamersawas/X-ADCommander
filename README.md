Installation

1- Download from https://github.com/msamersawas/X-ADCommander/releases 

2- unzip

3- copy the X-ADCommander folder to your preferred  module folder e.g. : C:\Program Files\WindowsPowerShell\Modules or C:\Users\Samer\Documents\WindowsPowerShell for PowerShell 5.1 or C:\Users\Samer\Documents\PowerShell for for PowerShell 7.0 and above

4- Run: import-Module X-ADCommander from a PowerShell console 

5- Run Start-XADCommander
